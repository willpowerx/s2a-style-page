# Adobe Clean fonts

Storybook now loads Adobe Clean from Adobe Fonts/Typekit (`https://use.typekit.net/hah7vzn.css`),
matching the Milo production experience. This directory remains gitignored so designers and
engineers can store personal copies of the OTF/WOFF files (e.g. for local design tooling),
but the Storybook build no longer consumes these files.
